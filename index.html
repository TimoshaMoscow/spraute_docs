Создавайте миры с помощью скриптов

Spraute Engine — мощная система скриптов для Minecraft, позволяющая создавать интерактивных NPC, сложные диалоги, квесты, кастомные интерфейсы, новые блоки, предметы и многое другое.

Создавайте NPC с моделями, текстурами, анимациями и сложным поведением

Динамический UI

Создавайте интерфейсы на серверной стороне с анимациями и обработчиками

Кастомные блоки и предметы

Создавайте свои блоки, предметы, руды и рецепты крафта прямо в скриптах

Java API

Взаимодействуйте с Java-кодом Minecraft напрямую из скриптов

Квестовая система
