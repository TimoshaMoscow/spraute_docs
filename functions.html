Пользовательские функции и Await

Пользовательские функции

fun name(a, b) {
    return a + b
}

# Вызов
val result = name(1, 2)

Работа с сущностями

healEntity(entity, amount)функция

Лечит сущность, не меняя max HP (в отличие от entity.hp = ..., которое выставляет и max, и текущее).

healEntity(player, 5)   # вылечить на 5 HP
heal(knight, 10)        # алиас
getPlayerFacing(entity)функция

Возвращает словарь направления взгляда (см. player.facing()). Работает для игрока и НИПа.

val facing = getPlayerFacing(player)
chat("Смотрит на: " + facing.direction)
getPlayerMotion(player)функция

Возвращает вектор движения (скорость) игрока в блоках за тик — список [vx, vy, vz] или null.

val motion = getPlayerMotion(player)
chat("Скорость по X: " + motion[0])
setPlayerMotion(player, vx, vy, vz)функция

Задаёт скорость игрока в блоках за тик. Можно передать список.

setPlayerMotion(player, 0, 0.5, 0)   # подбросить вверх
setPlayerMotion(player, [0.5, 0.2, -0.3])
# Алиасы: setPlayerMoveVector, set_player_motion

Поиск сущностей в радиусе

playersInRadius(x, y, z, radius)функция

Возвращает список игроков в сфере.

val players = playersInRadius(0, 64, 0, 10)
for p in players {
    chat(p.name + " рядом!")
}
playersNear(anchor, radius)функция

То же, но в качестве якоря — сущность, id NPC или строка.

val players = playersNear(readix_avatar, 5)
entitiesInRadius(x, y, z, radius, [filter])функция

Возвращает список сущностей в сфере. Фильтр: "any", "player", "npc", "mob", "living".

val entities = entitiesInRadius(0, 64, 0, 10, "npc")
val mobs = entitiesInRadius(0, 64, 0, 20, "mob")
entitiesNear(anchor, radius, [filter])функция

То же, но с якорем.

val entities = entitiesNear(player, 5, "living")

Async — асинхронные задачи

# Запуск с id
async "readix_idle" {
    readix_avatar.alwaysLookAt(knight)
    while (true) {
        await time(5)
        # действие каждые 5 секунд
    }
}

# Остановка
stop_task("readix_idle")

# Проверка завершения
if (taskDone("task_id")) {
    chat("Задача выполнена!")
}

# Ожидание
await task("readix_idle")

await uiClick внутри async не поддерживается.
