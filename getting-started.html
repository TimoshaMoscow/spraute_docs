Пошаговая инструкция

Текстовая версия гайда.

1. Установка и настройка

После установки движка в папку Ваша_Папка_Майнкрафт/spraute_engine/ запустите игру.

Для работы в скриптах используйте библиотеки. Например, импорт экономики:

import("economy")
ecoConfigure("minecraft:textures/item/gold_nugget.png", "coins")

2. Создание NPC

Создайте файл test.spr с таким содержимым:

create npc readix {
    name = "ReadixGG"
    hp = 20
    speed = 0.3
    model = "geo/defolt.geo.json"
    texture = "textures/entity/defolt.png"
    animation = "animations/npc_classic.animation.json"
    idleAnim = "idle"
    walkAnim = "idle"
    pos = 0, 64, 0
    rotate = 0, 0
    showName = true
    collision = true
}

3. Работа с игроком и безопасной точкой

Для получения первого игрока и поиска безопасной точки используйте:

val player = getFirstPlayer()
val spawn = findSafeBlock("player", 3, 1)

4. Настройка NPC

Заставьте NPC смотреть на игрока и отвечать на взаимодействие:

await interact(readix)
readix.alwaysLookAt(player)
say(readix, "Привет!")

5. Создание квеста

Создайте квест с целями:

add_quest_with_goals("kvest", "ZEMLYAA", readix.name, "зачем ему земля", list(quest_goal("земля?")))

При разрушении блока земли выполняется цель и спавнятся монеты:

on breakBlock("minecraft:dirt") -> h_break {
    zemlya = zemlya + 1
    if (zemlya >= 5) {
        complete_quest_goal("kvest", 0)
        set_quest_completed("kvest")
    }
    spawnOrb("minecraft:textures/item/gold_nugget.png", 2, _eventX, _eventY, _eventZ)
}

Полный код из видео: Все шаги объединены в один готовый скрипт в разделе Полные примеры.

Что такое Spraute Engine?

Spraute Engine — это система скриптов для Minecraft, написанная на языке .spr. Она позволяет создавать сложные интерактивные элементы без необходимости писать Java-код.

Spraute — это визуальное программирование! Вместе с движком поставляется IDE приложение Spraute Studio и плагин ProCode для расширенных визуальных блоков. Вы можете создавать сценарии без написания кода — буквально перетаскивая блоки, как в Scratch. Дизайн блоков выполнен в том же стиле, что и в Scratch.

Визуальное программирование

Spraute — это не только язык скриптов, но и полноценная среда визуальной разработки.

Spraute Studio

IDE-приложение для создания сценариев без кода. Перетаскивайте блоки, настраивайте параметры и сразу видите результат.

ProCode

Плагин для расширенных визуальных блоков. Добавляет дополнительные возможности для создания сложной логики.

Блоки как в Scratch

Дизайн блоков выполнен в стиле Scratch — интуитивно понятно даже новичкам. Просто соберите нужную логику из готовых элементов.

Не хотите писать код? Просто используйте Spraute Studio и собирайте сценарии из визуальных блоков! Код генерируется автоматически, а вы сосредотачиваетесь на логике.

Импорт функций из другого скрипта

# Запускаем скрипт в фоне
startScript("economy")

# Импортируем его функции
import("economy")

# Используем
ecoConfigure("minecraft:textures/item/gold_nugget.png", "coins")

startScript запускает скрипт, регистрируя его обработчики. import только импортирует функции, не запуская скрипт повторно.
