Кинематографическая камера

Общее описание

Управляет видом конкретного игрока (клиентская камера, не привязка к НИПу). После сцены вызывайте stopCamera(player).

Основные функции

setCamera(player, x, y, z)функция

Камера в точке; yaw/pitch — текущие у игрока.

setCamera(player, 10, 80, 10)
setCamera(player, x, y, z, yaw, pitch, holdTime, smoothTime)функция

Позиция и поворот (градусы). smoothTime = 0 — мгновенно.

setCamera(player, 10, 80, 10, 90, 0, 3, 0.8)
setCamera(player, x, y, z, yaw, pitch, holdTime, smoothTime, lockMovement, hideGui)функция

С блокировкой движения и скрытием HUD.

setCamera(player, 10, 80, 10, 90, 0, 3, 0.8, true, "minecraft")

Параметры: lockMovement (true/false), hideGui ("none", "minecraft", "all").

setCameraLookAt(player, camX, camY, camZ, lookX, lookY, lookZ, hold, smooth, track, lockMovement, hideGui)функция

Камера смотрит на координаты.

setCameraLookAt(player, 10, 80, 10, 0, 64, 0, 5, 1.0, true, true, "minecraft")
setCameraLookAt(player, camX, camY, camZ, entity, hold, smooth, track, lockMovement, hideGui)функция

Камера смотрит на сущность (игрока/НИПа/моба). track=true — слежение.

setCameraLookAt(player, 10, 80, 10, knight, 5, 1.0, true, true, "minecraft")
animateCamera(player, x, y, z, yaw, pitch, smoothTime, lockMovement, hideGui)функция

Плавно переместить активную камеру.

animateCamera(player, 20, 85, 15, 120, -10, 1.5)
animateCameraLookAt(player, x, y, z, target, smoothTime, track, lockMovement, hideGui)функция

Плавный перелёт + взгляд на цель.

animateCameraLookAt(player, 15, 85, 15, knight, 1.5, true)
stopCamera(player, smoothTime)функция

Плавно вернуть вид игроку. 0 — мгновенно.

stopCamera(player)      # за 0.5 сек
stopCamera(player, 0)   # мгновенно
resetCamera(player)функция

Мгновенно отключить камеру и вернуть управление.

resetCamera(player)
playCameraRoute(player, routeName, lockMovement, hideGui, returnSmooth)функция

Пролёт по сохранённому маршруту из spraute_engine/routes/. Возвращает длительность в секундах.

playCameraRoute(player, "intro")
await cameraRoute(player, routeName, lockMovement, hideGui, returnSmooth)await

То же, но скрипт ждёт окончания пролёта.

await cameraRoute(player, "intro", true, "minecraft", 0.8)

Декларативный синтаксис (camera id { ... })

camera intro {
    target = player
    pos = 100, 80, 200
    rotate = 45, -15
    lookAt = knight
    track = true
    time = 5
    smooth = true
    smoothTime = 1.0
    lockMovement = true
    hideGui = "minecraft"
}

Команды маршрута

/spraute route preview <имя>

Предпросмотр маршрута для себя

/spraute route list

Список сохранённых маршрутов

Маршруты создаются предметом Камера в креативе и сохраняются в spraute_engine/routes/<имя>.json.

Полный пример

# Установка камеры с плавным переходом
setCamera(player, 10, 80, 10, 90, 0, 3, 0.8, true, "minecraft")

# Камера смотрит на NPC с слежением
setCameraLookAt(player, 10, 80, 10, knight, 5, 1.0, true, true, "minecraft")

# Анимация перелёта
animateCamera(player, 20, 85, 15, 120, -10, 1.5)

# Проигрывание маршрута
playCameraRoute(player, "intro")

# Ожидание маршрута
await cameraRoute(player, "intro", true, "minecraft", 0.8)

# Возврат камеры
stopCamera(player)
resetCamera(player)
