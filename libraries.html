Встроенные библиотеки

Готовые решения для экономики, квестов и диалогов

Библиотеки достаточно положить в папку spraute_engine/scripts/ — они автоматически подхватываются. Импорт нужен только для использования функций в других скриптах.

Библиотека экономики — economy.spr

import("economy")импорт

Импортирует функции экономики.

ecoConfigure(texture, key)функция

Настраивает текстуру монет и ключ для сохранения.

ecoConfigure("minecraft:textures/item/gold_nugget.png", "coins")
ecoShowCoinsHud(player, texture)функция

Показывает HUD с количеством монет.

ecoShowCoinsHud(player, "minecraft:textures/item/gold_nugget.png")
ecoAddCoins(player, amount)функция

Добавляет монеты игроку.

ecoAddCoins(player, 5)
ecoOpenTrade(player, items, title, npcId)функция

Открывает окно торговли с NPC.

ecoOpenTrade(player, shop_ITEMS, "Лавка", "")
ecoSetCoinTexture(texture)функция

Меняет текстуру монет.

ecoSetCoinTexture("minecraft:textures/item/gold_nugget.png")

Формат товаров для торговли

val shop_ITEMS = list(
    dict("id", "minecraft:bread", "name", "Хлеб", "buy", 5, "sell", 2, "canBuy", true, "canSell", true)
)

Библиотека квестов — spraute_quests.spr

import("spraute_quests")импорт

Импортирует функции квестовой системы.

add_quest_with_goals(id, name, author, desc, goal_list)функция

Создаёт квест со списком целей.

add_quest_with_goals("kvest", "ZEMLYAA", "readix.name", "зачем ему земля", list(quest_goal("земля?")))
quest_goal(text)функция

Создаёт цель квеста.

quest_goal("Принести 30 слитков железа")
complete_quest_goal(id, index)функция

Отмечает цель квеста как выполненную (по индексу).

complete_quest_goal("kvest", 0)
set_quest_completed(id)функция

Отмечает весь квест как выполненный.

set_quest_completed("kvest")
set_quest_status(id, status)функция

Меняет статус квеста. Доступные статусы: QS_AVAILABLE, QS_PROGRESS, QS_COMPLETED, QS_FAILED, QS_FROZEN, QS_HIDDEN.

set_quest_status("kvest", QS_PROGRESS)
remove_quest(id)функция

Удаляет квест.

remove_quest("kvest")
has_quest(id)функция

Проверяет, существует ли квест с таким ID.

if (has_quest("kvest")) { ... }

Управление целями

# Добавить цель
add_quest_goal("kvest", "Собрать 5 блоков земли")

# Отметить цель как выполненную (по индексу)
complete_quest_goal("kvest", 0)

# Отметить цель как выполненную (по тексту)
complete_quest_goal_by_text("kvest", "земля")

# Провалить цель
fail_quest_goal("kvest", 0)

# Сбросить цель
reset_quest_goal("kvest", 0)

Работа с квестами

# Создать квест
add_quest("my_quest", "Мой квест", "Автор", "Описание", "Цель 1\nЦель 2\nЦель 3")

# Или с целями как список
add_quest_with_goals("my_quest", "Мой квест", "Автор", "Описание", list(quest_goal("Цель 1"), quest_goal("Цель 2")))

# Добавить описание
append_quest_desc("my_quest", "Дополнительная информация")

Библиотека диалогов — dialog_classic.spr

import("dialog_classic")импорт

Импортирует функции классического диалога.

dialogClassic(player, npc, question, answers)функция

Открывает диалоговое окно с вариантами ответов (до 3 вариантов). Возвращает ID выбранной кнопки.

val pick = dialogClassic(player, readix, "Что скажешь?", list("Да", "Нет", "Может"))
dialogPickIndex(pick)функция

Получает индекс выбранного ответа (0, 1, 2).

val idx = dialogPickIndex(pick)
dialogPickText(pick, answers)функция

Получает текст выбранного ответа.

val text = dialogPickText(pick, answers)
asyncиспользование

Диалог рекомендуется использовать внутри async блока для ожидания выбора.

async {
    val pick = dialogClassic(player, readix, "Что скажешь?", list("Да", "Нет"))
    val idx = dialogPickIndex(pick)
    if (idx == 0) {
        say(readix, "Ты выбрал Да!")
    }
}

Полный пример использования

import("economy")
import("spraute_quests")

# Настройка экономики
ecoConfigure("minecraft:textures/item/gold_nugget.png", "coins")

val player = getFirstPlayer()
ecoShowCoinsHud(player, "minecraft:textures/item/gold_nugget.png")

# Поиск безопасной точки для спавна NPC
val spawn = findSafeBlock(player, 7, 1)
val spawn_x = listGet(spawn, 0)
val spawn_y = listGet(spawn, 1)
val spawn_z = listGet(spawn, 2)

global val zemlya = 0

# Товары для магазина
val shop_ITEMS = list(
    dict("id", "minecraft:bread", "name", "Хлеб", "buy", 5, "sell", 2, "canBuy", true, "canSell", true)
)
val shop_TITLE = "Лавка"

# Создание NPC
create npc readix {
    name = "ReadixGG"
    hp = 20
    speed = 0.3
    model = "geo/defolt.geo.json"
    texture = "textures/entity/defolt.png"
    animation = "animations/npc_classic.animation.json"
    idleAnim = "idle"
    walkAnim = "idle"
    pos = spawn_x, spawn_y, spawn_z
    rotate = 0, 0
    showName = false
    collision = true
}

# Диалог с NPC
await interact(readix)
readix.alwaysLookAt(player)
say(readix, "Привет!")

# Создание квеста
add_quest_with_goals("kvest", "ZEMLYAA", readix.name, "зачем ему земля", list(quest_goal("земля?")))
readix.playOnce("happy")

await interact(readix)
say(player, "Привет!")
readix.stop("happy")

# Открытие магазина
on interact(readix) -> h_interact {
    ecoOpenTrade(player, shop_ITEMS, shop_TITLE, "")
}

# Сбор земли и квест
on breakBlock("minecraft:dirt") -> h_break {
    zemlya = zemlya + 1
    if (zemlya >= 5) {
        complete_quest_goal("kvest", 0)
        set_quest_completed("kvest")
    }
    # Спавн монет при ломании земли
    spawnOrb("minecraft:textures/item/gold_nugget.png", 2, _eventX, _eventY, _eventZ)
}
